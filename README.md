# ScamShield Pro – AI Scam Message Detector

Final year Diploma project (Computer Science & Engineering) – 2026  
Detects Indian scams like UPI KYC fraud, digital arrest, lottery/refund scams, government scheme frauds using TF-IDF + PyTorch neural network.

## Features
- Real-time message classification (single or batch)
- 90.91% test accuracy
- Clean professional dark-mode UI
- PDF export of results
- Flask web application

## Tech Stack
- Backend: Python, Flask, PyTorch, scikit-learn (TF-IDF)
- Frontend: Bootstrap 5, Font Awesome, jsPDF
- Model: Neural network binary classifier
