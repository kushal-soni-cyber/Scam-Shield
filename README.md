
# ScamShield - AI Scam Message Detector

**Capstone Project**  
**AICTE Batch-7**  
**IBM AI/ML Internship**

A smart, real-time web application that detects Indian scam messages such as UPI KYC fraud, Digital Arrest scams, Lottery/Refund scams, and Government scheme frauds using **TF-IDF + PyTorch Neural Network**.

---

## 🎯 Live Demo - 
**Deployed Link:** [Visit Now](...)  

---

## ✨ Features
- Real-time classification of single or multiple messages
- Clean professional interface
- Batch processing (one message per line)
- Confidence score with color coding
- One-click copy message
- Export results as PDF
- Responsive design (works perfectly on mobile and laptop)

---

## 📊 Model Performance
- **Test Accuracy:** **90.91%**
- Trained for 50 epochs
- Final Training Loss: **0.0034**

---

## 🛠️ Tech Stack
- **Backend:** Python, Flask, PyTorch, scikit-learn (TF-IDF)  
- **Frontend:** Bootstrap 5, Font Awesome, jsPDF  
- **Deployment:** Render.com (Free)

---

## 🚀 How to Run Locally

1. Clone the repository
```bash
git clone https://github.com/YOUR-USERNAME/scamshield-pro.git
cd scamshield-pro
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
python main.py
```

4. Open in browser: http://127.0.0.1:5000/
